知识图谱可视化使用指南

探索知识关系

通过可视化形式了解知识点之间的关联关系，帮助理解学习路径。

互动式探索

点击、拖动、缩放，以不同方式与知识图谱交互，深入了解知识结构。

个性化学习

收藏关键知识点，构建自己的学习路径，提高学习效率。

基本操作指南

图谱导航

  • 拖动图谱：点击空白区域并拖动可移动整个图谱
  • 放大缩小：使用工具栏中的放大/缩小按钮或鼠标滚轮
  • 重置视图：点击重置按钮回到初始视图
  • 移动节点：点击并拖动节点可调整其位置

节点交互

  • 查看详情：点击任意节点显示其详细信息
  • 查看关系：在节点详情中可查看与其他知识点的关系
  • 收藏节点：在节点详情面板中点击"添加到收藏"
  • 深入探索：点击"查看节点图谱"进入以该节点为中心的视图
小提示

对于复杂的知识图谱，可以使用不同的视图模式来聚焦于特定内容：

  • 使用单元视图查看特定单元内的知识点关系
  • 使用节点视图专注于特定知识点及其关联
  • 使用学习路径视图查看推荐的学习顺序
